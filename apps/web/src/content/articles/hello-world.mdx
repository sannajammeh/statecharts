## Code Examples

Here's a simple state machine definition:

```typescript
const toggleMachine = {
  initial: "inactive",
  states: {
    inactive: {
      on: { TOGGLE: "active" }
    },
    active: {
      on: { TOGGLE: "inactive" }
    }
  }
};
```
